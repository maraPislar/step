School Simulation

This project was created to simulate a school: add courses, teachers, certificates, etc. Courses have a number of free spaces for students, as well as days in which they can be completed (be aware of weekends). Teachers have job specifications that permit them to teach a course. Students can get certificates when they complete a course and can't enroll in courses where they already have certificates. When a student has certificates for all the courses, he finishes the school. The aim is for all the students to finish all courses. Programming language used: Java.
